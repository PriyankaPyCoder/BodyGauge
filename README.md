# BMI Calculator 🧮

This is a simple BMI (Body Mass Index) Calculator built using Python.
It takes user input for weight (kg) and height (feet) and calculates BMI
along with the health category.

---

## Features
- User input from terminal
- Height conversion (feet to meters)
- BMI calculation
- Health category result

---

## Technologies Used
- Python

---

## How to Run
1. Install Python
2. Run the file using:
   python BMI_Calculator.py
3. Enter weight and height when asked
