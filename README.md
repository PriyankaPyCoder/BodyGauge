## BodyGauge 🧮

A Python-Based BMI Calculator for Health Assessment

BodyGauge is a simple terminal-based Python application that calculates Body Mass Index (BMI) using user-provided weight and height. The program converts height from feet to meters, computes BMI, and displays the corresponding health category.

---

## 🚀 Features

- Interactive terminal-based BMI calculator
- Accepts weight in kilograms and height in feet
- Automatic height conversion (feet → meters)
- BMI calculation with health classification
- Input validation for positive values
- Handles invalid input using exception handling
- Option to calculate BMI multiple times

---

## 🛠️ Tech Stack

- Python (Core Python)
- Conditional Statements
- Exception Handling
- Loops

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Download or clone this repository
3. Run the program using:
   python bodygauge.py
4. Enter your weight and height when prompted
5. View your BMI result and health category

---

## 📊 BMI Categories Used

- Underweight: BMI < 18.5
- Normal weight: 18.5 – 24.9
- Overweight: 25 – 29.9
- Obesity: BMI ≥ 30
